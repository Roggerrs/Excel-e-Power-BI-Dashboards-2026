# 📊 Criando Relatórios Dinâmicos com uso de Parâmetros no Power BI

Projeto desenvolvido durante o bootcamp da DIO com foco em criação de dashboards dinâmicos utilizando parâmetros no Power BI.

---

# 🚀 Objetivo do Projeto

Construir um relatório dinâmico capaz de alterar categorias e métricas de análise através de parâmetros, permitindo maior interatividade e flexibilidade na visualização dos dados.

---

# 🛠️ Tecnologias Utilizadas

- Power BI Desktop
- DAX
- Parâmetros de Campo
- Visualizações Dinâmicas
- Navegação entre páginas
- Storytelling com dashboards

---

# 📌 Funcionalidades

✅ Página Home personalizada  
✅ Navegação entre páginas com botões  
✅ Dashboard financeiro interativo  
✅ Gráficos dinâmicos  
✅ Uso de parâmetros para troca de categorias  
✅ Uso de parâmetros para troca de métricas  
✅ Storytelling visual  
✅ Layout customizado

---

# 📈 Indicadores Utilizados

- Receita
- Lucro
- Margem
- COGS
- Sales

---

# 🔄 Parâmetros Dinâmicos

O projeto permite alterar dinamicamente:

## Categorias
- Country
- Product
- Segment
- Month Name

## Métricas
- Sales
- COGS
- Profit

---

# 📷 Prévia do Dashboard

## Home
Dashboard inicial com navegação interativa.

## Relatório
Página com análises financeiras e indicadores principais.

## Análise Dinâmica
Página com gráficos alterados dinamicamente através de parâmetros.
