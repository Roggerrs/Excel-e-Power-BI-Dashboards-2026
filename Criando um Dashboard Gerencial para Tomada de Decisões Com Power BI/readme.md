# 📊 Dashboard Financeiro com Power BI

Projeto desenvolvido em Power BI com foco em análise financeira, visual moderno e experiência do usuário (UX).

---

# 🎯 Objetivo

Criar um dashboard financeiro interativo utilizando Power BI, aplicando:

- análise de vendas
- visualização de dados
- navegação entre páginas
- experiência do usuário
- organização visual
- interatividade

---

# 🛠 Tecnologias Utilizadas

- Power BI
- DAX
- Visualizações interativas
- Navegação entre páginas
- Botões personalizados

---

# 📌 Funcionalidades

## 🏠 Página Home
- tela inicial personalizada
- botão “Explorar análise”
- navegação para relatório

---

## 📈 Página Relatório
- tabela comparativa por país
- gráfico de vendas por ano
- gráfico de vendas por país
- visual moderno em tema roxo

---

## 📄 Página Extra
- análise complementar
- navegação entre páginas
- experiência de usuário aprimorada

---

# 🔄 Navegação

O projeto possui navegação interativa entre páginas:

- Home → Relatório
- Relatório → Página 1
- Página 1 → Home
- Página 1 → Relatório

---

# 🎨 Design

O dashboard foi desenvolvido com foco em:

- contraste visual
- organização dos elementos
- proporção visual
- experiência do usuário
- identidade visual moderna
