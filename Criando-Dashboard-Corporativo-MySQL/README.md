# Criando Dashboard Corporativo com Integração MySQL

## Sobre o Projeto

Projeto desenvolvido no Power BI com integração de banco de dados relacional, com foco em análise corporativa de funcionários, departamentos e projetos.

O dashboard apresenta:

- Total de funcionários
- Total de gerentes
- Total de departamentos
- Funcionários por departamento
- Funcionários por gerente
- Horas trabalhadas por projeto

## Tecnologias Utilizadas

- Power BI
- SQL
- SQLite / MySQL
- Modelagem de Dados
- Power Query
- DAX

## Sobre o Azure

Inicialmente o projeto seria integrado ao Azure SQL Database. Porém, devido à exigência de cartão de crédito para ativação do serviço, foi utilizada uma alternativa local com banco relacional (SQLite/MySQL), mantendo os mesmos conceitos de integração, consultas e visualização de dados.

## Objetivo

Demonstrar capacidade de:

- Integração entre banco de dados e BI
- Tratamento de dados
- Modelagem relacional
- Construção de dashboards executivos
- Visualização estratégica
