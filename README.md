# 📊 Dashboard de Vendas - Power BI

## 📌 Sobre o Projeto

Este projeto foi desenvolvido utilizando o Power BI com o objetivo de analisar dados de vendas de forma visual e intuitiva.

O dashboard apresenta indicadores importantes que auxiliam na tomada de decisão, permitindo identificar padrões de vendas, desempenho por país e distribuição de lucro.

---

## 🎯 Objetivo

Criar um relatório interativo que permita:

* Analisar o volume de vendas por país
* Avaliar a quantidade de produtos vendidos
* Identificar o lucro por região
* Visualizar a distribuição de lucro por segmento

---

## 📈 Visuais Desenvolvidos

### 🥧 Gráfico de Pizza

* **Lucro (Profit) por Segmento**
* Permite identificar quais segmentos geram maior retorno

### 🗺️ Mapa 1

* **Soma de Sales e Units Sold por Country**
* Exibe volume de vendas e quantidade vendida por país

### 🗺️ Mapa 2

* **Soma de Profit por Country**
* Mostra onde estão os maiores lucros

---
