# 🍦 Sistema Sorveteria BI

Dashboard desenvolvido no Power BI para análise de vendas de uma sorveteria, utilizando gráficos interativos, indicadores visuais e identidade visual personalizada.

![Dashboard Sorveteria BI](https://raw.githubusercontent.com/Roggerrs/Excel-e-Power-BI-Dashboards-2026/main/Sistema_sorveteria%20BI/2026-05-18T20-53-33.198Z.png)

---

## 📊 Sobre o Projeto

O objetivo deste projeto foi criar um dashboard simples, visual e intuitivo para acompanhar informações de uma sorveteria, como:

- 💰 Total faturado
- 🍨 Sabores mais vendidos
- 👨‍💼 Atendentes ativos
- 📈 Faturamento por atendente
- 🍧 Distribuição por tamanho de sorvete

O dashboard foi desenvolvido com foco em:

- Organização visual
- Experiência do usuário
- Uso de cores suaves
- Elementos gráficos personalizados
- Storytelling visual

---

## 🎨 Layout do Dashboard

O projeto utiliza:

- Fundo rosa claro
- Menu lateral roxo
